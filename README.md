## JAMN

A hyper-minimalist notepad designed for focused writing. JAMN provides a clean, typewriter-inspired canvas with essential formatting tools and a distraction-free interface.

---

### Features

- **Multiple Themes:** Choose from a variety of curated color palettes to suit your environment.
- **Multiple Fonts:** Select between different classic serif, sans-serif, and monospace fonts.
- **Text Highlighting:** Dynamic markers that contrast perfectly with your active theme.
- **Headlines:** Quick formatting for H1, H2, and H3 titles.
- **Lists:** Easy bullet points and numbered list support.
- **Text Colors:** Custom color palette for specific text styling.
- **Instant Inversion:** Toggle between light and dark modes instantly.
- **Mobile Ready:** Install as a web app with a hideable toolbar for mobile writing.
- **Save as .txt:** Export your notes directly to your device as plain text.

---

### Keyboard Shortcuts

- **Alt + 1/2/3:** Headlines (H1, H2, H3)
- **Alt + U / O:** Bullet or Numbered Lists
- **Alt + H:** Toggle Highlighting
- **Alt + C:** Change Text Color
- **Alt + F:** Change Font
- **Alt + D:** Change Theme
- **Alt + I:** Invert Theme
- **Alt + B:** Show/Hide Toolbar
- **Ctrl + B/I/U:** Bold, Italics, Underline
- **Win/Cmd + Shift + H:** Show All Shortcuts
